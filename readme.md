# 概要
このレポジトリはVSCodeを利用して自動サンプルテストケース&手入力値テスト実行するために作成されたものです。利用例としてabc197でテストケースを実行できます。

# 使い方
## ファイル名を変更する。
problemsディレクトリ以下のファイル名は問題のコンテスト名に変更する。

## ログインする。
oj login -u <username> -p <password> "https://atcoder.jp/"

## test/以下のディレクトリのテストケースを実行する。
ctrl + shift + b

## input.txtからテストを実行する。
fn + F5
